# Biblioteca de Jogos

Uma biblioteca digital moderna para você visualizar e organizar sua coleção de jogos. O projeto foi focado em oferecer uma navegação rápida, visual elegante e que funciona bem em qualquer aparelho (computador ou celular).

## ✨ O que o app oferece

- **Visual Moderno:** Design em modo escuro com efeitos de transparência e brilho.
- **Navegação Suave:** O deslizar da página é fluido e agradável.
- **Menu Prático:** Lateral organizada para filtrar jogos por categorias e acompanhar o progresso.
- **Carregamento Elegante:** Uma tela de entrada animada enquanto os dados são preparados.
- **Pronto para Celular:** Menu adaptado e visual otimizado para telas menores.

## 🛠 O que usei

- **Base do Projeto:** [Next.js 15](https://nextjs.org/)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
- **Estilização:** CSS puro para um visual leve e rápido.
- **Ícones:** Font Awesome
- **Fluidez:** Lenis para o movimento suave da tela.

## ⚙️ Como rodar na sua máquina

1. Faça o download ou clone o projeto.
2. No terminal, instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o projeto:
   ```bash
   npm run dev
   ```
4. Abra `http://localhost:3000` no seu navegador.

## 🌐 De onde vêm os dados?
Os jogos são buscados automaticamente de uma base de dados externa.
